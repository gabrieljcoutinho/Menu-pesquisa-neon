# Search Vibe - Clean Cyber

Interface de busca de alta performance com estética **Cyberpunk Minimalista**, focada em interatividade visual avançada e feedback de estado via CSS3 moderno.

---

## 🛠 Tecnologias
* **HTML5** (Estrutura semântica)
* **CSS3** (Custom Properties, Flexbox, Keyframes)

---

## ⚡ Diferenciais Técnicos

### 1. Arquitetura de Feedback Visual
O componente utiliza o seletor `:focus-within` para transformar o container pai quando o input é acessado. Isso permite uma transição de escala (`scale 1.02`) e uma mudança dinâmica na **Glitch Bar** lateral, que altera sua cor e intensifica o brilho (glow) para indicar atividade.

### 2. Sistema de Animações
* **Pulse Glow:** Ciclo de animação `pulseGlow` que altera o brilho e a opacidade dos elementos de destaque.
* **Fade-In Entry:** Entrada suave dos elementos com deslocamento no eixo Y para uma sensação de profundidade.

### 3. Responsividade Adaptativa
O componente foi projetado com uma estratégia de adaptação de escala para garantir a integridade visual em dispositivos móveis.
